# IT Women Esbjerg

This is the official website for IT Women Esbjerg, built to share information about our events, initiatives, and community.

## Features

- Event listings and details
- Newsletter signup (integrated with Brevo)

## Tech Stack

- **Hugo**: Static site generator
- **Tailwind CSS**: Utility-first CSS framework
- **JavaScript**: For interactive features (newsletter, gallery, calendar)
- **Brevo (Sendinblue)**: Newsletter integration
- **GitHub Pages**: Hosting and deployment

## Development

- Configuration files: `hugo.toml`, `config.development.toml`
- Main content in `content/`, layouts in `layouts/`, static assets in `static/`
- To run locally: `hugo server -D` or `make dev`
- To build for production: `hugo`
